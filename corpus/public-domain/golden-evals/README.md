# corpus/public-domain/golden-evals/

Frozen regression artifacts — one per work in the active 8-work calibration corpus.

## Purpose

These JSON files are the canonical "expected behavior" targets for the public-domain corpus.
They are generated by running RevisionGrade's full long-form evaluation pipeline against each
clean source text, then saving and freezing the result.

## What they verify

Every time evaluation pipeline logic changes (Pass 3 synthesis, QualityGateV2, chunking,
scoring, long-form rendering), run the regression suite against these golden files to confirm:

- Pride and Prejudice does not receive a low dialogue score
- Dracula is not penalized for epistolary structure
- Great Expectations is not flagged as lacking narrative closure
- The Yellow Wallpaper does not fail for short-form compression
- The Time Machine is not dinged for absent romantic subplot
- All 8 complete long-form evaluation without timeout
- All 8 produce scored 13-criterion output
- Evidence is preserved across Pass 1 → Pass 2 → Pass 3

## Naming convention

```
<work-id>.dream-longform.json
```

## Status

Golden eval files are populated when the long-form pipeline is stable enough to produce
repeatable scored output on the full-manuscript route. Placeholder stubs live here until
the first real run is committed.

## Canon rule

Do not overwrite a golden eval file without a deliberate regression-comparison step.
The file comparison goal is stable *judgment*, not identical wording.
