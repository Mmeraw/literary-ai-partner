# Author Bio Template v2

## Purpose

Reusable author biography template for Agent Readiness package generation.

This v2 template separates bio outputs by submission context instead of treating `author_bio` as one generic product.

## Absolute rules

- Author Bio is fact-governed.
- Never invent credentials, publications, awards, residences, degrees, platform, personal history, identity, or professional expertise.
- Prefer manuscript-relevant authority over generic literary credentials.
- Connect the author to the selected manuscript when possible.
- Do not include every credential just because it exists.
- Do not use placeholders in final output.

---

# Bio Types

## 1. Query Letter Bio

```txt
id: query_letter_bio
target_length: 75-130 words
purpose: close a query letter with concise credibility
```

### Template

```txt
[Author Name] is a [professional/lived-experience identity] whose work [connects to manuscript themes/category]. [He/She/They] write[s] [genre/category] about [core subject/lane]. [Location/regional/professional authority sentence tied to THIS manuscript]. [Optional single proof point: credential, project, startup, public platform, publication, or related work if supplied and relevant].
```

### Use when

- Bio appears inside a query letter.
- Word count is tight.
- The story pitch should remain dominant.

---

## 2. Agent Submission Bio

```txt
id: agent_submission_bio
target_length: 120-190 words
purpose: standalone submission package bio or long agency form
```

### Template

```txt
[Author Name] ([optional rank/credential if supplied and useful]) is a [professional/lived-experience identity] whose fiction explores [author lane]. [Location/regional/professional sentence connecting author to selected manuscript].

Beyond [selected manuscript category], [Author Name]'s work includes [related projects/author brand if supplied]. [Credential/proof-point sentence if relevant]. [Optional closing line with current writing location or brand statement if supplied].
```

### Use when

- Agent form asks for a biography.
- The package includes a separate author bio section.
- More credentials and related projects help market positioning.

---

## 3. QueryTracker / Short Form Bio

```txt
id: short_form_submission_bio
target_length: 50-85 words
purpose: quick credibility under strict field limits
```

### Template

```txt
[Author Name] is a [primary identity] turned author. [His/Her/Their] work examines [core themes/lane]. [He/She/They] write[s] [category] between/from [locations if supplied]. [Optional projects/proof point in one sentence].
```

### Use when

- Form fields are short.
- QueryTracker or similar tools impose limits.
- Low-age-signal phrasing is preferred.

---

## 4. Website / Press Bio

```txt
id: website_press_bio
target_length: 250-450 words
purpose: author website, media kit, book page, press materials
```

### Template

```txt
[Author Name] is an author whose work explores [large thematic lane]. A [professional/lived-experience background], [he/she/they] bring[s] [perspective] to stories about [recurring subjects].

[Paragraph expanding author brand, literary themes, and recurring story worlds.]

[Paragraph naming major projects and how they fit together.]

[Closing sentence tying the author's work to a clear reader-facing promise or thematic question.]
```

### Use when

- Branding matters more than compression.
- Bio lives on website, press kit, author page, or book page.
- Atmospheric positioning is appropriate.

---

# Forbidden language

- "Aspiring author."
- "Has always loved stories."
- "Hopes readers will connect."
- "Uniquely qualified" unless clearly supported.
- Unsupported awards/publications/representation.
- Résumé bullet dumping in query-letter bio.
- Placeholder brackets.

# Quality gate

A bio is not ready unless it passes all five questions:

1. Are all facts supplied or verified?
2. Does the bio fit the requested context and length?
3. Does it connect the author to this manuscript or author lane?
4. Does it avoid generic self-description?
5. Would an agent understand why this author is credible without feeling oversold?
