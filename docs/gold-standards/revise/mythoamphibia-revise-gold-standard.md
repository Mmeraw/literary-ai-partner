# The Lost World of MythOAmphibia — Revise Gold Standard Fixture

Status: gold-standard Revise fixture v1  
Benchmark family: multi-species mythic eco-fantasy / object POV / frog-human civilization / relic memory  
Runtime role: calibration fixture for six-part Revise diagnostics and A/B/C repair candidates.

## Fixture principle

A MythOAmphibia Revise card must preserve multi-species logic, frog/human boundary pressure, object consciousness, relic memory, ecological collapse, mythic doctrine, and child/witness accessibility. It must not flatten the manuscript into generic fantasy worldbuilding.

## Gold-standard card example

### Card title

Clarify object-consciousness POV without breaking wonder or canon.

### Source evidence

Use manuscript-local evidence around a story-bearing object, relic, tool, shelter object, garment, or inherited item with POV/domain consciousness. The evaluator must cite the exact passage available in the run; this fixture supplies card shape, not replacement evidence.

### Six-part diagnostic

| Part | Gold-standard content |
|---|---|
| Evidence | Anchor to the exact object-consciousness moment, including the object’s sensory register, memory function, or influence on nearby characters. |
| Symptom | The object POV may be imaginative but risks confusing readers if its agency, limits, or memory function are not clear. |
| Cause | The manuscript may treat object consciousness as surprise or flavor before establishing the rule: what can the object know, remember, want, or influence? |
| Fix direction | Add a compact rule-bearing beat that shows one limit and one power of the object’s consciousness without turning the scene into lore explanation. |
| Reader effect | The reader accepts object POV as governed magic/mythic consciousness rather than random whimsy. |
| Mistake-proofing | Do not over-rationalize the object. Do not make every object omniscient. Do not erase comic wonder or childlike strangeness. |

### Repair goal

Make object POV feel governed, emotionally purposeful, and story-bearing while preserving the manuscript’s wonder.

## Candidate repairs

### Candidate A — Conservative repair

Add one sentence inside the object’s POV that defines a boundary: it remembers pressure, smell, touch, owner history, ritual charge, or fear—but not abstract human intention.

### Candidate B — Balanced repair

Add a short beat where the object misreads a human action according to its own material logic. This clarifies consciousness through limitation and keeps the scene alive.

### Candidate C — Bold repair

Create a recurring object-consciousness rule across two scenes: first, the object stores a sensory imprint; later, that imprint alters a character’s choice, fear, or recognition. The payoff should be subtle, not deus ex machina.

## Acceptance criteria

A repaired passage passes if:

- the object has a clear sensory or memory domain;
- the object is not omniscient;
- the repair supports plot, character, or mythic doctrine;
- the tone remains wondrous, strange, or funny where appropriate;
- no new canon contradictions are introduced;
- the repair does not overexplain the object’s magic.

## Risk notes

- Do not turn object POV into technical exposition.
- Do not make object agency unlimited.
- Do not erase the comic or childlike register.
- Do not contradict accepted relic/object consciousness rules.

## Withhold examples

Withhold a card if it says only:

- "Clarify the object POV."
- "Explain the magic system."
- "Make the relic more important."
- "Add worldbuilding."

Those are too generic for Workbench release.

## What this teaches Revise

This fixture teaches Revise to preserve governed strangeness: object consciousness must become clearer without becoming ordinary, overexplained, or detached from character pressure.
