## Part 7 — What the Pipeline Must Learn

The fit-gap reveals the following systemic failures that RevisionGrade's pipeline must be trained to avoid:

### Failure Mode 1: Symbolic-System Flattening
**What happened:** The pipeline read the river as setting/mood rather than as an agentive system of memory, judgment, and archive. With the symbolic engine invisible, Thematic Integration and Symbolic System Integrity were unscored or underscored.
**Root cause hypothesis:** Environmental/atmospheric prose is a high-frequency "setting" association; the model defaulted to classifying river passages as description rather than as a recurring agency system tied to plot events (disappearances, banks, trucks, weather, dogs).
**Gate to add:** Any evaluation of this manuscript that treats the river only as setting/atmosphere — without recognizing it as an agentive symbolic system — must be rejected.

### Failure Mode 2: Animal-Sensory Layer Dismissed as Decoration
**What happened:** Fritz and Schultz were treated as pet/companion charm. The truth target identifies them as a functional sensory layer that tests spaces before humans can interpret them.
**Root cause hypothesis:** Animals in fiction are a high-frequency "decorative/emotional" association; the reduce pass filed dog scenes as warmth rather than as evidence/stakes.
**Gate to add:** Any evaluation that recommends removing or reducing the dogs, or treats them only as decoration, must be rejected.

### Failure Mode 3: Community Flattening
**What happened:** Smokehouse Camp — the book's richest arena and moral center — was reduced to a generic Indigenous-community setting, erasing protocol density (cake covenant, bark maps, council, food/work/social texture).
**Root cause hypothesis:** Without a community-protocol extractor, the dense, distributed camp material was summarized as "setting" rather than analyzed as a structural lane.
**Gate to add:** Any evaluation that does not treat Smokehouse Camp as a structural lane (with William, Esmé, Anthony, cake covenant, bark maps, witness/ownership) must be rejected.

### Failure Mode 4: Cultural / Protocol Gate Omission
**What happened:** The offending eval produced no cultural/protocol risk section, made no sensitivity-reader recommendation, and implicitly certified the manuscript as closer to ready than it is.
**Root cause hypothesis:** Risk analysis is not a default evaluation section; without a mandatory gate, the model optimized for craft scoring and skipped representational risk.
**Gate to add:** For manuscripts depicting fictionalized real-world-adjacent cultural material, a cultural/protocol risk section is mandatory and readiness is capped until external review is recommended.

### Failure Mode 5: Over-Evidence Mis-Diagnosed as Strength
**What happened:** The investigation/ledger layer was praised as documentary strength. The truth target identifies over-evidence as the manuscript's principal drag.
**Root cause hypothesis:** Volume of detail is a high-frequency "thoroughness" association; the model rewarded accumulation rather than diagnosing it as pacing drag.
**Gate to add:** Evaluations must distinguish evidentiary scale from pacing drag; recommending MORE evidence for this manuscript is a failure condition.

### Failure Mode 6: Recommendation to Add Already-Present Elements
**What happened:** The offending eval recommended adding a corporate antagonist, establishing a POV, and adding evidence — all already present (or the inverse of what's needed).
**Root cause hypothesis:** The model generated generic "improvement" recommendations without checking presence against the canon ledger.
**Gate to add:** Any "add/introduce/establish" recommendation for a present element must be reclassified as `ALREADY_PRESENT` and blocked.
