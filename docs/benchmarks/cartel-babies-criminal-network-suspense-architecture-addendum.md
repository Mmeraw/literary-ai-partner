# Cartel Babies — Criminal Network & Suspense Architecture Addendum

## Purpose

This addendum teaches RevisionGrade how to evaluate criminal ecosystems, cartel pressure systems, abduction narratives, coercion, survival decision-making, institutional corruption, inherited violence, and suspense architecture.

Core rules:

1. Suspense is uncertainty under consequence.
2. Fear and danger are not the same thing.
3. Criminal networks operate as systems, not villains.
4. Coercion changes decision quality.
5. Survival logic must be evaluated separately from morality.
6. Family inheritance can function as narrative pressure.

## Layer 19 — Criminal Network Architecture

Track:
- hierarchy
- operators
- intermediaries
- facilitators
- corruption points
- economic incentives

Calibration rule:
A cartel is a system of relationships, not merely an antagonist.

## Layer 20 — Coercion & Power Layer

Evaluate:
- threats
- leverage
- dependency
- intimidation
- reputation pressure
- hostage dynamics

Calibration rule:
Power is often exercised without violence.

## Layer 21 — Abduction & Captivity Layer

Track:
- loss of control
- uncertainty
- information restriction
- escape opportunities
- psychological adaptation

Calibration rule:
Captivity changes how characters process risk.

## Layer 22 — Survival Decision Architecture

Track:
- available options
- perceived options
- actual constraints
- decision quality
- outcome quality

Calibration rule:
Good decisions can still produce bad outcomes.

## Layer 23 — Institutional Corruption Layer

Track:
- compromised institutions
- silence
- incentives
- jurisdictional gaps
- accountability failures

Calibration rule:
Corruption is a pressure system, not a background detail.

## Layer 24 — Inherited Violence Layer

Track:
- family history
- cultural normalization
- generational consequences
- identity pressure

Calibration rule:
Violence can be inherited socially without being inherited morally.

## Layer 25 — Suspense Architecture Layer

Required progression:

curiosity → concern → fear → uncertainty → consequence → revelation

Calibration rule:
Suspense depends on what might happen, not what already happened.

## Calibration Requirements

A compliant evaluation must identify:

- criminal network structure
- coercion systems
- hostage or captivity pressure
- survival logic
- corruption architecture
- inherited violence themes
- suspense escalation quality
- consequence management

## Failure Conditions

An evaluation is incomplete if it:

- reduces the cartel to a single villain;
- ignores coercion dynamics;
- evaluates survival choices only through morality;
- ignores institutional failure;
- confuses action with suspense;
- misses inherited violence as a pressure system.
