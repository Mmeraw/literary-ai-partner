# Revise Voice Preservation Standard

**Authority:** Revision Governance Volume VII  
**Scope:** Voice, POV, tense, rhythm, and character-language protection.

RevisionGrade must preserve:
- Narrative voice
- POV
- Tense
- Character knowledge
- Character vocabulary
- Speech pattern
- Narrative rhythm
- Register and tonal discipline

## Permitted Changes
- Clarify a sentence while preserving voice.
- Reduce confusion while preserving author identity.
- Improve pacing without replacing style.

## Forbidden Changes
- Replace author voice with generic model prose.
- Add sentimental, cinematic, or melodramatic beats unsupported by context.
- Change POV distance or character knowledge.
- Insert dialogue inconsistent with the character’s speech pattern.

Failure code: `VOICE_DRIFT`.
