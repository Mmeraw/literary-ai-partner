# Revise Canon Preservation Standard

**Authority:** Revision Governance Volume VII  
**Scope:** Factual, continuity, relationship, world-rule, and manuscript-canon preservation.

RevisionGrade must preserve:
- Character identity
- Timeline
- Location
- Relationships
- World rules
- Established facts
- Known motivations and knowledge boundaries

## Forbidden Changes
A revision may not introduce unsupported:
- New names
- New locations
- New dates
- New numbers
- New relationships
- New motivations
- New plot facts

Failure code: `CANON_DRIFT` or `UNSUPPORTED_FACT`.
